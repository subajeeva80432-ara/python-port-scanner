🚀 Just built my first Cyber Security mini project!

I created a simple **Port Scanner using Python** to understand how systems detect open ports on a network.

🔍 What the project does:
• Scans a target IP address
• Checks ports 1–100
• Identifies whether any ports are open
• Displays "No ports are open" if none are detected

💡 What I learned:
• Basics of **network sockets in Python**
• How **port scanning works in cybersecurity**
• Understanding **open vs closed ports**
• Importance of **network security and attack surface**

This project helped me understand one of the **first steps in security testing – scanning and reconnaissance**.

Looking forward to building more **Cyber Security and Python projects**.
